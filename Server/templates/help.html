How to plan a trip using e-tinerary ?

  1. Home Page Search



    • The first step to create your trip is to input some data like the destination, the travel dates and the number of persons.

    • To create a trip to best match your preferences, we need to know more about your trip. For that you should navigate to the "Plan a trip" tab or click on the "More options" button.

  2. Plan a Trip Search



    In this form, besides the general information about the trip, you also can input some other fields, such as:

    • Constraint the maximum budget for the trip (Yes / No)
    • Input the maximum budget for the trip
    • Pick the travel mode for the trip
    • Choose among the interests provided the ones that suits the best your preferences
